# Insurance ML Project

A machine learning project for insurance data analysis and prediction using Python and a web-based interface.

## Project Overview

This project contains a machine learning model trained on insurance data, with both a Jupyter notebook for analysis and a web interface for interaction.

## Project Structure

- **Insurance.ipynb** - Jupyter notebook containing the machine learning model development, training, and analysis
- **demarrer.py** - Python script to start/run the project
- **test_Insurance.csv** - Test dataset for model evaluation
- **train_Insurance.csv** - Training dataset for model development
- **index.html** - Web interface (frontend)
- **script.js** - JavaScript logic for the web interface
- **styles.css** - Styling for the web interface

## Requirements

- Python 3.x
- Jupyter Notebook
- Required Python packages (see notebook for dependencies)

## Getting Started

### Option 1: Jupyter Notebook

Open and run the notebook to explore the model development:

```bash
jupyter notebook Insurance.ipynb
```

### Option 2: Run the Application

Start the application using:

```bash
python demarrer.py
```

Then open `index.html` in your web browser to access the web interface.

## Datasets

- **train_Insurance.csv** - Training data used to build and train the machine learning model
- **test_Insurance.csv** - Testing data used to evaluate model performance

## Features

- Machine learning model for insurance prediction
- Web-based user interface for predictions
- Data analysis and visualization in Jupyter notebook

## Usage

1. Train/explore the model using the Jupyter notebook (`Insurance.ipynb`)
2. Run the application with `demarrer.py`
3. Access the web interface via `index.html`
4. Make predictions or analyze insurance data through the UI
